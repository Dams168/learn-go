# Package Embed di Go

Go 1.16 memperkenalkan fitur baru yang disebut "embed" yang memungkinkan kita untuk menyisipkan file atau direktori ke dalam binary Go. Fitur ini sangat berguna untuk menyimpan aset seperti file konfigurasi, gambar, atau bahkan file teks langsung di dalam aplikasi kita tanpa perlu mengakses file sistem saat runtime.

Untuk menggunakan fitur embed, kita perlu mengimpor paket "embed" dan menggunakan direktif `//go:embed` untuk menentukan file atau direktori yang ingin kita sisipkan.

Dalam Penggunaan secara nyata, embed berfungsi ketika kita ingin menyimpan file konfigurasi atau aset lainnya di dalam aplikasi kita tanpa perlu mengakses file sistem saat runtime. Dengan embed, kita dapat menyisipkan file tersebut langsung ke dalam binary aplikasi kita, sehingga memudahkan distribusi dan penggunaan aplikasi tanpa perlu khawatir tentang keberadaan file eksternal. Misalnya, kita dapat menyisipkan file konfigurasi seperti `config.json` ke dalam aplikasi kita menggunakan embed, sehingga kita dapat mengaksesnya langsung dari kode tanpa perlu membaca file dari disk. Ini sangat berguna untuk aplikasi yang dijalankan di lingkungan yang terbatas atau ketika kita ingin memastikan bahwa semua aset yang diperlukan sudah termasuk dalam aplikasi kita. Selain itu, embed juga memungkinkan kita untuk menyisipkan direktori, sehingga kita dapat menyimpan beberapa file sekaligus di dalam aplikasi kita. Dengan menggunakan embed, kita dapat dengan mudah mengelola aset dan konfigurasi aplikasi kita tanpa perlu khawatir tentang keberadaan file eksternal saat runtime.
Namun, perlu diingat bahwa file yang disisipkan menggunakan embed akan menjadi bagian dari binary aplikasi kita, sehingga ukuran binary dapat meningkat tergantung pada ukuran file yang disisipkan. Oleh karena itu, sebaiknya kita hanya menyisipkan file yang benar-benar diperlukan dan mempertimbangkan ukuran file yang akan disisipkan agar tidak membuat aplikasi kita terlalu besar.
